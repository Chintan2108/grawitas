Help

This web app is meant solely for the purpose of demonstrating our Wikipedia talk page parser that is submitted to the Demo Track at EACL 2017. Please do not share the url to this website as the traffic on this server is limited. If accepted the web app will be moved to a different server.

This demo only includes the core parser component, which means that the input has to be a valid talk page in the Wikipedia markup syntax. A possible workflow is explained below. For a quick try-out click the Generate Example Input to use some included demo talk pages.

Workflow

Step 1:

Go to Wikipedia.org and pick an article.

Step 2:

In the article click on Talk in the upper left menu, then click on Edit in the upper right. Now copy the syntax of the talk page from the large textarea in the center.

Step 3:

Paste the syntax copied in step 2 into the left textarea of Grawitas. Optionally pick a different output format in the top right corner.
